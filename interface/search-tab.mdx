## Overview

![SCR-20250509-njfo-2.png](/images/SCR-20250509-njfo-2.png)

<Steps>
  <Step title="Search mode switch">
    Switch between `Visual` and `Speech` search
  </Step>
  <Step title="Main search box">
    This is the main search box where you enter your search query.
  </Step>
  <Step title="Additional search methods">
    There are a few additional methods of searching. The exact behavior of these depend on the parent NLE.

    <Tabs>
      <Tab title="Premiere Pro">
        <Card title="Match Source Monitor" horizontal>
          Takes the frame currently visible in the Source Monitor and performs a search with it.
        </Card>
        <Card title="Match Timeline Monitor" horizontal>
          Takes the frame currently visible in the Timeline Monitor and performs a search with it.
        </Card>
        <Card title="Match Pasteboard" horizontal>
          Takes the image currently in your <Tooltip tip="">pasteboard</Tooltip> and performs a search with it.
        </Card>
      </Tab>
    </Tabs>
  </Step>
  <Step title="Search outside project">
    If switched on, Jumper will search in any analyzed media you have, even if its not in the currently open project.
  </Step>
</Steps>

## Search result view

After performing a search, the results are shown in a grid below the search box.

![SCR-20250509-nply-4.png](/images/SCR-20250509-nply-4.png)

 1. Search history
 2. Clear Search history
 3. Go back to the previous search result page
 4. Change insertion mode
 5. Adjust thumbnail size
 6. The file in which the scene is found
 7. Timestamp of start of scene
 8. Timestamp of end of scene
 9. Show file in project
10. Mark In/Out
11. Add scene to timeline
12. Find similar
